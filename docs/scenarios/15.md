# Scenario 15: Pipeline Scheduling

> Pipelines SHALL be able to run on a schedule

## Implementation

`pipeline.with_schedule(schedule)` attaches a schedule to any pipeline. The `scheduler.gleam` module provides matching logic.

### Schedule Types (`src/thingfactory/types.gleam:178-199`)

- `NoSchedule` — on-demand only
- `Daily(hour, minute)` — daily at specific UTC time
- `Weekly(day_of_week, hour, minute)` — weekly
- `Monthly(days, hour, minute)` — specific days of month
- `Interval(interval_ms)` — every N milliseconds
- `Cron(expression)` — cron expression format

### Scheduler API (`src/thingfactory/scheduler.gleam`)

- `matches_now(schedule, timestamp_ms)` — check if schedule triggers
- `interval_ready(interval_ms, elapsed_ms)` — check interval elapsed
- `description(schedule)` — human-readable schedule text
