# Scenario 42: Runnable Multi-Step Demo Pipeline

> The project SHALL have runnable demos that run a multi-step pipeline that
> compiles a project & runs tests.

## Implementation

Multiple demo pipelines compile projects and run tests:

- `gleam_build_pipeline()` — runs `gleam check`, `gleam test`,
  `gleam format --check`, `gleam build --target javascript`,
  `gleam build --target erlang`
- `dogfood_pipeline()` — builds the entire thingfactory project (Gleam + web)
  with parallel execution
- `typescript_build_pipeline()`, `rust_build_pipeline()`, `go_build_pipeline()`
  — language-specific build workflows

All are runnable via CLI: `gleam run -m thingfactory/cli -- run gleam`
