# Scenario 29: GUI Log Viewing with Streaming

> The Pipeline GUI SHALL allow for the viewing of logs efficiently, streaming from the providing agents.

## Implementation

### StepLogViewer (`web/components/run/StepLogViewer.tsx`)

Renders step log output with syntax highlighting and status indication. Displays per-step logs on the run detail page.

### Real-Time Streaming

`GET /api/runs/[runId]/stream` provides server-sent events for real-time run updates. The `RunDetailClient.tsx` component subscribes to this stream.

### Evidence

| Artifact | Description |
|---|---|
