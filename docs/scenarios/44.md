# Scenario 44: GUI Artifact Download

> The Pipeline GUI SHALL provide a way to download produced artifacts.

## Implementation

### ArtifactsList Component (`web/components/run/ArtifactsList.tsx`)

Displays artifacts produced by a pipeline run with download buttons. Each artifact shows name, size, and creation time.

### API Endpoints

- `GET /api/runs/[runId]/artifacts` — list artifacts for a run
- `GET /api/artifacts/[id]` — download a specific artifact

### Database Schema

`web/db/migrations/0004_artifacts.sql` — artifacts table with id, run_id, name, content, created_at.
