# Scenario 2: Rapid CLI Installation and Example Execution

> A user SHALL be able to rapidly install the CLI and run it against an example
> pipeline hosted in a file in this project's examples.

## Implementation

Install is 3 commands (`git clone`, `cd`, `gleam build`). Running an example
pipeline is one command. 26 example pipelines are available in
`src/thingfactory/examples.gleam`.

### CLI Commands

```bash
gleam run -m thingfactory/cli -- list           # list 15 runnable pipelines
gleam run -m thingfactory/cli -- run basic       # run a pipeline
gleam run -m thingfactory/cli -- run parallel -c # compact output
```
