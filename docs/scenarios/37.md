# Scenario 37: CLI Artifact Extraction

> The CLI SHALL offer a way to extract produced artifacts from a run.

## Implementation

The `-o/--output-dir` flag extracts artifacts to disk after pipeline execution.

### CLI Flag (`src/thingfactory/cli.gleam:71-76`)

```bash
gleam run -m thingfactory/cli -- run artifacts -o ./output
```

### Extraction Logic (`src/thingfactory/cli.gleam:468-503`)

`extract_artifacts(result, output_dir)` iterates over `result.artifacts` dict, writes each key as a file in the output directory. Reports count and per-file status.
