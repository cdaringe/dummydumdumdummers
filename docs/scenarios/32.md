# Scenario 32: No Embedded Examples in Binary

> The CLI SHALL NOT embed examples pipelines in the binary or compiled artifact.

## Implementation

Example pipelines are defined as Gleam functions in `src/thingfactory/examples.gleam`. They are compiled as regular code (not embedded data) and loaded at runtime via function calls. The CLI references them by name through `execute_pipeline()`.

The binary contains compiled function references, not embedded pipeline data (no JSON, YAML, or serialized pipeline definitions baked in).

### Evidence

| Artifact | Description |
|---|---|
| `src/thingfactory/examples.gleam` | Pipelines as functions, not embedded data |
| `src/thingfactory/cli.gleam:108-131` | Pipeline lookup by name → function call |
| `bin/cli.mjs` | Entry point imports compiled module |
