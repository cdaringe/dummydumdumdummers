# Scenario 46: Minimalist GUI Styling

> The Pipeline GUI SHALL stylistically focus on minimalism with light grays,
> whites, tonal grays. All highlights should be primarily light & dark greens.
> It should feel flat, simple, mechanical. Backgrounds should be textured with
> subtle patterns.

## Implementation

### Global Styles (`web/app/globals.css`)

CSS variables define the color palette:

- Backgrounds: light grays, whites, tonal grays
- Highlights: light and dark greens (--color-primary)
- Typography: clean, minimal font sizing
- Subtle background textures for visual depth
- Flat, mechanical aesthetic — no shadows or gradients

### Components

All components use CSS variables for consistent theming:

- `StatusBadge` — green for success, red for failure
- `DurationBadge` — monospace, minimal
- `Sidebar` — flat navigation with green active indicator
- Cards, tables — flat borders, no shadows
