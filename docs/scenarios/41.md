# Scenario 41: Async, Parallel, and Accumulating Examples

> The project SHALL host examples that have asynchronous steps, parallel steps, and steps that accumulate values and pass them to later steps.

## Implementation

### Parallel Steps

- `parallel_build_pipeline()` — clone → lint || test → build → package
- `parallel_multi_target_pipeline()` — diamond pattern with parallel branches

### Accumulated Values

- `artifact_sharing_pipeline()` — steps generate and pass config/secrets/build results
- `queue_worker_pipeline()` — producer enqueues items, worker pulls and processes all

### Inter-Step Communication

- `simple_messaging_pipeline()` — pub-sub message passing
- `multi_topic_messaging_pipeline()` — multi-topic coordination
- `event_driven_pipeline()` — event-driven workflow coordination
