# Scenario 40: Dogfooding Pipeline

> The system SHALL be built & tested, dogfooding itself.

## Implementation

`dogfood_pipeline()` in `examples.gleam` builds thingfactory using its own
pipeline engine.

### Pipeline Structure (`src/thingfactory/examples.gleam:822-869`)

```
gleam_check ──────┐
gleam_format ─────┼─→ gleam_build_js ──┐
                  │   gleam_build_erl ──┼─→ verify
web_install ──────┼─→ web_build ────────┘
```

Uses real commands via `command_runner.step()`:

- `gleam check`, `gleam format --check`
- `gleam build --target javascript`, `gleam build --target erlang`
- `npm --prefix web install`, `npm --prefix web run build`

Parallel execution via `parallel_executor.execute_parallel()`.
