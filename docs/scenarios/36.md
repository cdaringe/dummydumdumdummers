# Scenario 36: Compact and Verbose CLI Modes

> The CLI SHALL show compact and verbose modes of progress. Compact mode shows step-n/of-m, verbose shows all logs in real time.

## Implementation

### Three Output Modes (`src/thingfactory/cli.gleam:37-41`)

- **Verbose** (default) — prints step start/finish with `>>` prefix, full status, duration
- **Compact** (`-c`) — prints `[N/M] step_name ✓ duration` per step completion
- **Interactive** (`-i`) — verbose output + REPL drilldown after execution

### Progress Callbacks

```
compact_progress():  [1/3] fetch ✓ 0ms
verbose_progress():  >> [1/3] fetch
                        ✓ OK (0ms)
```

### Interactive Mode (`src/thingfactory/interactive_cli.gleam`)

After pipeline execution, enters REPL with commands: `help`, `list`, `step <N>`, `stats`, `artifacts`, `exit`.
