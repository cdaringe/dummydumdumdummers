# Scenario 21: Multi-Language Pipeline Authoring

> Pipelines SHOULD be able to be authored in a variety of languages, but Gleam SHOULD be the first-class reference.

## Implementation

Gleam is the first-class pipeline authoring language. The system compiles to both JavaScript and Erlang targets, enabling:

- **Gleam** (first-class) — type-safe pipeline builder with compile-time checks
- **JavaScript** — pipelines can call JS functions via Gleam FFI (`@external(javascript, ...)`)
- **Erlang** — pipelines can call Erlang functions via Gleam FFI (`@external(erlang, ...)`)
