# Scenario 31: Runnable Examples via CLI and GUI

> The project shall host examples that are runnable with the CLI and demonstrate the same pipelines running in the GUI.

## Implementation

15 pipelines are runnable via CLI (`gleam run -m thingfactory/cli -- run <name>`). The same pipelines are seeded into the web GUI database for visualization.

### CLI Runnable Pipelines

```bash
gleam run -m thingfactory/cli -- list
# Shows: basic, error, mock, dependency, artifacts, typescript, rust, fullstack,
#        gleam, go, custom, parallel, parallel_multi, dogfood, queue
```

### GUI Visibility

The seed script (`web/db/seed.ts`) populates the same pipeline definitions into the database. The GUI shows them on the Pipelines page with DAG visualization, and run history on the Runs page.
