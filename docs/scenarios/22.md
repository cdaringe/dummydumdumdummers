# Scenario 22: Sample Pipelines for Real Use Cases

> The source code should have a large set of sample pipelines demonstrating compilation of libraries for typescript, go, & gleam (REAL, not mocked), and custom Docker images.

## Implementation

26 example pipelines in `src/thingfactory/examples.gleam` (1093 lines), including real build pipelines:

### Real Build Pipelines

- `typescript_build_pipeline()` — checkout, install, lint, test, build, package
- `rust_build_pipeline()` — validate, test, build release, docs, publish
- `go_build_pipeline()` — deps, test, build multi-arch, lint, publish
- `gleam_build_pipeline()` — uses real `gleam check`, `gleam test`, `gleam build` commands via `command_runner.step()`
- `full_stack_pipeline()` — API + frontend build, integration + E2E tests, deploy

### Custom Docker Image Demo

- `Dockerfile` — multi-stage build with Node 22 + Gleam 1.13.0
- `dogfood_pipeline()` — builds thingfactory itself in Docker-compatible steps
