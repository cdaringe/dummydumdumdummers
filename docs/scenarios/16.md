# Scenario 16: External Event Triggers

> Pipelines SHALL be able to be triggered by external events

## Implementation

`pipeline.with_trigger(trigger)` configures webhook-based triggering. `webhook_trigger.gleam` provides event creation, matching, and deduplication.

### Trigger Types (`src/thingfactory/webhook_trigger.gleam`)

- `NoTrigger` — manual only
- `WebhookTrigger(matcher)` — triggers on matching webhook events
- `ManualTrigger` — explicit manual trigger

### Matchers

- `TopicMatcher(topic)` — match by event topic
- `SourceMatcher(source)` — match by event source
- `AnyMatcher(matchers)` — OR composition
- `AllMatcher(matchers)` — AND composition
- `CustomMatcher(fn)` — user-defined predicate

### Built-in Helpers

- `on_github_push()`, `on_github_pr()`, `on_github_event()`
- `github_push_event()`, `github_pr_event()`, `gitlab_push_event()`, `custom_event()`
- Deduplication via `DeduplicationState` with configurable max age
