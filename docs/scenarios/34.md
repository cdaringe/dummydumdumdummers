# Scenario 34: Gantt/Timeline View

> The Pipeline GUI SHALL provide a gantt/timeline view of Pipeline runs such
> that users can easily see which steps are taking the most time.

## Implementation

### GanttTimeline Component (`web/components/run/GanttTimeline.tsx`)

Horizontal bar chart showing step execution timelines:

- Each step renders as a colored bar proportional to its duration
- Steps sorted by execution sequence
- Duration labels on each bar
- Color-coded by status (success/failed/skipped)
- Visually identifies bottleneck steps

### Integration

Displayed on the run detail page (`web/app/runs/[runId]/page.tsx`) using step
trace data from the database.
