# Scenario 25: One Worker Per Core Default

> The default runner host should optimistically try to initialize one worker per
> available core.

## Implementation

`runner_host.new()` calls `get_cpu_count()` via FFI to detect logical CPU cores
and creates one worker per core (minimum 1).

### Runner Host (`src/thingfactory/runner_host.gleam:96-108`)

```gleam
pub fn new() -> RunnerHost {
  let cores = get_cpu_count()
  let workers = int.max(cores, 1)
  RunnerHost(worker_count: workers, ...)
}
```

FFI backends: `os.cpus().length` (JavaScript),
`erlang:system_info(logical_processors)` (Erlang).
